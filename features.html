Fonctionnalites Completes

Explorez en detail toutes les capacites de MatchPro IA pour transformer votre processus de recrutement.

Algorithme de Matching Hybride + Moteur Jokil Pre-Entraine

Notre algorithme combine trois approches complementaires AVEC un moteur ML deja entraine sur 10000+ matchings reels.

  • 40% Score Semantique - Sentence-BERT paraphrase-multilingual-mpnet (local)
  • 40% Regles Metier - Competences, experience, localisation, teletravail
  • 20% Score Predictif ML Jokil - XGBoost pre-entraine sur 10000+ cas reels
  • Predictions avec 85%+ de confiance des le premier jour
  • Apprentissage continu adapte a votre usage specifique
  • Filtres par score minimum (50%, 70%, 85%, 90%)
  • Export CSV des resultats avec scores detailles

Analyse Enrichie Candidat

Lors de l'import d'un CV, MatchPro IA extrait automatiquement des informations avancees pour enrichir le profil et optimiser les matchings.

  • Niveau de seniorite: Junior / Confirme / Senior / Expert (detecte automatiquement)
  • Secteur dominant: Analyse des experiences pour identifier le secteur principal
  • Metier principal: Identification du poste cœur via dictionnaire Jokil
  • Score d'employabilite: Calcul base sur competences, stabilite, polyvalence
  • Stabilite du parcours: Tendance stable / mobile / en transition
  • Polyvalence: Detection profils specialises vs multi-secteurs
  • Recommandations: Suggestions de types d'offres a cibler

Analyse Enrichie Offre

Lors de la creation d'une offre, MatchPro IA extrait automatiquement toutes les informations importantes depuis la description.

  • Remuneration: Salaire min/max, periode (annuel/mensuel/horaire), brut/net, variables
  • Teletravail: Jours/semaine, pourcentage, full remote, modalites
  • Avantages: Tickets resto, mutuelle, prevoyance, 13e mois, primes, CE, formation
  • Competences: Techniques + soft skills avec niveau et caractere obligatoire
  • Experience: Annees min/max, diplome requis, certifications
  • Details poste: Urgence, date prise poste, taille equipe, management, horaires
  • Langues: Detection automatique des langues requises + niveau

Parsing CV Automatise par IA

Importez n'importe quel CV et laissez l'IA extraire automatiquement toutes les informations pertinentes.

  • Formats supportes: PDF, DOCX, Images (OCR Tesseract)
  • Extraction: nom, email, telephone, adresse
  • Experiences professionnelles detectees
  • Competences techniques et soft skills
  • Formations, diplomes et langues
