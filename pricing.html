Questions Frequentes

Comment recuperer une cle API ?

OpenAI: platform.openai.com/api-keys | Anthropic Claude: console.anthropic.com | Google Gemini: aistudio.google.com/apikey | Mistral: console.mistral.ai | Ollama: Installer depuis ollama.com (pas de cle requise, IA 100% locale)

Comment configurer Ollama pour fonctionner 100% hors-ligne ?

1. Telecharger Ollama depuis ollama.com | 2. Installer et lancer Ollama | 3. Telecharger un modele (ex: ollama pull llama3.2) | 4. Dans MatchPro IA → Parametres → API → Ajouter Ollama (endpoint: http://localhost:11434) | 5. Activer et mettre en priorite haute. Aucune donnee ne quittera votre PC.

Dois-je payer les cles API en plus de l'abonnement ?

Oui, vous fournissez vos propres cles API (OpenAI, Claude, etc.) et payez directement aux fournisseurs selon votre usage. MatchPro IA ne facture QUE l'abonnement logiciel. Alternative gratuite: Utilisez Ollama (IA locale, zero cout API).

Le machine learning s'ameliore-t-il vraiment avec mon usage ?

Oui ! Le moteur Jokil est livre pre-entraine sur 10000+ matchings reels. Il est deja performant des le jour 1. Puis chaque matching accepte/refuse enrichit le modele pour l'adapter a VOS criteres specifiques (secteur, exigences, contexte). Le re-entrainement est automatique.

Mes donnees sont-elles partagees avec les fournisseurs IA ?

Parsing CV et analyse: Les donnees sont envoyees au provider IA choisi (OpenAI, Claude, etc.) pour extraction. Matching: 100% local (Sentence-BERT + regles + ML XGBoost), aucune donnee envoyee. Avec Ollama: Tout est local, zero donnee partagee.

Difference entre plan Candidat et Entreprise ?

Candidat (9.99€/mois): 1 profil, ideal recherche d'emploi, offres illimitees. Entreprise (49.99€/mois): Candidats/offres illimites, API France Travail, analytics complets, ideal recruteurs/agences. 2 mois d'essai gratuits pour tester sans risque.

Puis-je upgrader ou changer de plan ?

Oui, a tout moment ! Passez de Candidat a Entreprise ou Sur Mesure quand vous voulez. Toutes vos donnees (candidats, offres, matchings, modele ML) sont conservees. Contactez-nous pour la migration.
