Deux Modes de Fonctionnement

MatchPro IA s'adapte a votre usage: recherche d'emploi ou recrutement professionnel.

Mode Candidat

  1. Import de votre CV (PDF/DOCX/Image avec OCR)
  2. Parsing automatique + Analyse Joskil (seniorite, metier, employabilite)
  3. Creation de votre profil enrichi dans la base locale
  4. Import d'offres d'emploi qui vous interessent
  5. Lancement du matching candidat → offres
  6. Prediction ML de la compatibilite pour chaque offre
  7. Resultats tries par score (semantique + regles + ML)

Cle API sollicitee: Uniquement lors du parsing CV et analyse. Pas utilisee pour le matching si Ollama local configure.

Mode Entreprise/Recruteur

  1. Creation d'offres d'emploi + Analyse Joskil (extraction salaire, teletravail, avantages)
  2. Import de CV candidats (manuel, France Travail API, ou sources externes)
  3. Parsing + Analyse automatique de chaque candidat
  4. Lancement du matching offre → candidats
  5. Prediction ML du taux de reussite pour chaque candidat
  6. Filtrage par score minimum (50%, 70%, 85%, 90%)
  7. Export CSV des meilleurs profils pour analyse RH

Cle API sollicitee: Parsing CV, analyse offre/candidat. Pas utilisee pour matching (Sentence-BERT local) sauf si extraction donnees complexes.

Confidentialite & Utilisation des Cles API

Transparence totale sur l'utilisation de vos donnees et de vos cles API.

Quand la Cle API est Sollicitee

  • ✓ Parsing CV (extraction texte → donnees structurees)
  • ✓ Analyse candidat (seniorite, metier, employabilite)
  • ✓ Analyse offre (extraction salaire, avantages)
  • ✓ Si vous choisissez un provider cloud (OpenAI, Claude, Gemini, Mistral)

Quand la Cle API N'est PAS Utilisee

  • ✓ Matching candidat-offre (Sentence-BERT local)
  • ✓ Calcul des scores semantiques (embeddings caches)
  • ✓ Application des regles metier (100% local)
  • ✓ Predictions ML XGBoost (modele local)
  • ✓ Si Ollama configure (IA 100% locale)

Donnees Restant sur Votre PC

  • ✓ Base SQLite locale (candidats, offres, matchings)
  • ✓ Modele ML XGBoost entraine (ml_model.pkl)
  • ✓ Donnees d'entrainement (training_data.json)
  • ✓ Cache des embeddings
  • ✓ Fichiers CV importes
  • ✓ Toutes vos cles API (chiffrees AES-256)

Mode 100% Hors-Ligne avec Ollama

En configurant Ollama (IA locale), AUCUNE donnee ne quitte votre ordinateur. Le parsing CV, l'analyse et le matching fonctionnent entierement en local. Ideal pour les donnees sensibles ou usages sans connexion internet.
